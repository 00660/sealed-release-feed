# sealed-release-feed

```bash
curl -fsSL https://github.com/00660/sealed-release-feed/raw/refs/heads/main/install.sh | sudo bash -s --
```

指定网络授权 API：

```bash
curl -fsSL https://github.com/00660/sealed-release-feed/raw/refs/heads/main/install.sh | sudo bash -s -- --license-api-url https://your-license-api.example
```

指定面板端口：

```bash
curl -fsSL https://github.com/00660/sealed-release-feed/raw/refs/heads/main/install.sh | sudo bash -s -- --web-port 8790
```
